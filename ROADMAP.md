# PrismForge Long-Term Roadmap

This roadmap captures the long-term product vision for PrismForge. It is meant to keep the big picture aligned while current implementation stays focused on the MVP.

No functional code should be built from this document yet. Treat this as strategic direction, not an active engineering spec.

---

## Near-Term Strategic Notes

### Crawler Accuracy and Frequency

- Make the crawler much more accurate.
- Make the crawler run much more frequently.
- Loosen processing parameters so high-confidence findings can move through more easily.
- Allow findings with **85%+ confidence** through regardless of stricter review gates, assuming they pass basic safety and relevance checks.

### Level Progression

- Make levels harder to progress through.
- Leveling should feel meaningful, earned, and exciting.
- XP should reward real progress, not shallow clicking.

- An AI planner
- An AI startup advisor
- A one-time business idea generator

The stronger direction is:

> An AI co-founder that discovers opportunities, builds products, launches them, watches the market, and improves them over time.

That changes the value proposition from:
